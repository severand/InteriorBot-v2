# InteriorBot v2

Telegram-бот для генерации дизайна интерьера с использованием AI (text-to-image).

## Описание

Вторая версия бота InteriorBot с новым функционалом генерации дизайна интерьера на основе текстового описания (без загрузки фотографий).

## Установка

1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Создать файл `.env` с необходимыми ключами
4. Запустить: `python bot/main.py`
